# Parlement de Gérionie

Application web de simulation parlementaire construite avec Next.js. par Yahia Ben Letaief

## Prérequis

Avant de commencer, assurez-vous d’avoir installé :

- [Node.js](https://nodejs.org/) version 22.12 ou supérieure
- [Git](https://git-scm.com/)

## Cloner le dépôt

Vous pouvez cloner le dépôt avec Git.

### Via SSH
```bash
git clone git@github.com:Yahiabenlet/interfaceAssemblee.git
```

### Via HTTPS
```bash
git clone https://github.com/Yahiabenlet/interfaceAssemblee.git
```

Ensuite, placez-vous dans le dossier du projet :

```bash
cd interfaceAssemblee
```

## Installation

Installez les dépendances du projet avec :

```bash
npm install
```

## Lancer le projet

Démarrez le projet en local avec :

```bash
npm run dev
```

Puis ouvrez votre navigateur à l’adresse suivante :

```bash
http://localhost:3000
```
